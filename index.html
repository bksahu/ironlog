SETUP

Iron Log needs a free Supabase account to store your data. Follow these steps:

STEP 1 — Create Supabase project
Go to supabase.com → Sign up free → New Project → wait ~1 min for it to start.

STEP 2 — Create the database table
In Supabase: SQL Editor → New query → paste & run this:


STEP 3 — Enable Google login
Supabase → Authentication → Providers → Google → toggle ON → paste your Google Client ID & Secret (from Google Cloud Console → APIs & Services → Credentials).

STEP 4 — Add your site URL
Supabase → Authentication → URL Configuration:
• Site URL: https://YOURUSERNAME.github.io/REPONAME
• Redirect URLs: same URL above

STEP 5 — Get your API keys
Supabase → Project Settings → API → copy Project URL and anon public key below.
